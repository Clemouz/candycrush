j'ai commencé à partir du squelette du candy crush fourni dans le fichier TP1;
de ce squelette, j'ai ajouté les implémentations nécéssaires
c'est-à-dire : détection des alignements, puis implémentation des chutes (avec réactions en chaine) et score incrémental.
La partie détection des alignements est assurée par les fonction findLineAt et tagLine
L'implémentation des chutes est assurée par le fichier "Falling.mjs"
le score est géré dans le fichier "Delete.mjs" par les fonctions setScore, getScore, setScoreMultiplier, getScoreMultiplier et addScore.

Des voies d'améliorations sont les suivantes :
-ajouter des soundeffets
-créer des niveaux de difficultés
-activer le timer
